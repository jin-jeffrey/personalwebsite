# Personal Website
A personal website built using React. Template available to all.

## Quickstart

1. Go into personalwebsite folder
```bash
$ cd personalwebsite
```
2. Install the node packages
```bash
$ npm install
```
3. Run application 
```bash
$ npm start
```
The site should be running on [http://0.0.0.0:3000](http://0.0.0.0:3000)\
